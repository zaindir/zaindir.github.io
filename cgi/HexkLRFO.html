How to Reset an Canon Mg2570 Printer

  1. Press and hold the Stop button and the Power button on the printer at the same time.
  2. While still holding the Power button, press the Stop button 5 times.
  3. Release the Stop button, then release the Power button. The printer will enter Service (Safe) Mode.
  4. Prepare some paper and place it into the printer paper tray.
  5. Download the Service Tool application for resetting the Canon MG2570 printer.
  6. Open the Service Tool application.
  7. Go to the Main tab menu.
  8. Select Main in the Clear Ink Counter section.
  9. Click the Set button on the right side of the Clear Ink Counter section.
  10. Select Main again in the Ink Absorber Counter section.
  11. Click the EEPROM button in the Print section; the printer will start printing a report.
  12. Wait until the printing process is complete; the printer is now ready to be used.
