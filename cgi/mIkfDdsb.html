How to Reset an Canon iP1980 Printer

  1. Turn on the printer that will be reset.
  2. Press the Power button to turn off the printer.
  3. Unplug the power cable from the back of the printer after the printer is completely powered off.
  4. Press and hold the Power button and do not release it.
  5. While still holding the Power button, reconnect the power cable to the printer. The power LED light will turn on.
  6. Press the Reset button on the printer twice while still holding the Power button.
  7. Release the Power button; the printer will enter standby mode.
  8. Make sure the printer is connected to the computer using a USB printer cable.
  9. Prepare blank paper and place it into the printer paper tray.
  10. Download the Canon IP1980 Resetter application.
  11. Extract the downloaded resetter application files.
  12. Open the GeneralTool file located inside the resetter folder.
  13. Select Yes if a User Account Control notification appears; the General Tool application will open.
  14. Click the Device ID button to display printer information.
  15. Check the options Cleaning and EEPROM Clear.
  16. Click the Main button.
  17. Click the Platen button below it.
  18. Click the EEPROM Dump button; an EEPROM Dump window will appear.
  19. Click the Quit button to close the EEPROM Dump window.
  20. Click the Heat Test Pattern button; the printer will automatically activate and print a test pattern.
  21. Click the Quit button to close the resetter application.
  22. Turn off the printer by pressing the Power button.
  23. Unplug the power cable from the back of the printer even if the power LED is still on or blinking.
  24. Reconnect the power cable.
  25. Turn the printer back on; the printer should now return to normal and be ready for regular printing.
