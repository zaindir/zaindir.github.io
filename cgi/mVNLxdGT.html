How to Reset an Epson L4150/L4160 Printer

  1. Connect the printer to a laptop or computer.
  2. Turn on the printer.
  3. Then, disable the antivirus software.
  4. Extract the downloaded file.
  5. After extracting the RAR file, you will find two main directories: app and key.
  6. Open the app directory.
  7. Point the cursor to AdjProg L4150-L4160, right-click it, and select Run as Administrator. An error message will appear showing your Hardware ID. Leave this message window open, as the Hardware ID is displayed there.
  8. Open the key directory, then open the EXE subdirectory.
  9. Right-click the file WLGen_L4150 L4160 RSTFULL and select Run as Administrator to open the Key Generator program.
  10. In the program, select License Manager and then click Add License.
  11. In the Customer field, enter any name, for example aa. Repeat the same process for the Company field using any name.
  12. Enter the Hardware ID shown in the previous error message (from step 7) into the Hardware ID field.
  13. Save the settings by clicking Save, then create the License Key by clicking Create License Key and confirm by clicking OK. A new directory will be created using the customer and company names you entered, for example aa-aa.
  14. Open the newly created directory (for example aa-aa) and copy all files inside it.
  15. Paste those files into the directory where Adjprog is located, which is the app directory.
  16. Right-click AdjProg L4150-L4160, select Run as Administrator, and click OK when the WinLicense message appears. The resetter program will then open.
  17. Click the Particular Adjustment Mode menu.
  18. Scroll down until you find the Waste Ink Pad Counter option under the Maintenance section.
  19. Click OK; the Waste Ink Pad Counter window will open.
  20. Check the Main Pad Counter and Platen Pad Counter options.
  21. Click the Check button; the printer usage points or percentages will be displayed.
  22. Check the Main Pad Counter again, while the Platen Pad Counter option may be left unchecked.
  23. Click the Initialize button.
  24. Click OK if an Information dialog box appears; another Information dialog will appear instructing you to turn off the printer.
  25. Turn off the printer.
  26. Wait a few moments.
  27. Turn the printer back on.
  28. Click OK once again.
  29. Click the Finish button.
