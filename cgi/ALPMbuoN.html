How to Reset an Epson L1300 Printer

  1. Download the resetter program for the Epson L1300 printer series.
  2. Extract the downloaded RAR file.
  3. Open the AdjProg file in the extracted folder.
  4. Click the Select menu; the Select window will open.
  5. Click the Model Name field and choose the printer series you are using; in this case, Epson L1300.
  6. Click the Port field to select the printer port; choose the port that shows your printer series.
  7. Click OK; the Select window will close.
  8. Click the Particular Adjustment Mode menu at the bottom right; a window with various adjustment and maintenance options will appear.
  9. Click Waste Ink Pad Counter under the Maintenance section.
  10. Click OK; the Waste Ink Pad Counter window will open.
  11. Check the box next to Main Pad Counter.
  12. Click the Check button under Check the counter value; the main pad counter value will be displayed.
  13. Check the Main Pad Counter box again.
  14. Click the Initialize button under Initialize the selected counters.
  15. Click OK to confirm when the Information dialog appears.
  16. Click OK again if a message instructs you to turn off the printer.
  17. Turn off the printer by pressing and holding the power button until the power indicator light turns off.
  18. Wait a few moments, then turn the printer back on; if the printer indicator light is normal, the reset process is complete.
  19. Close all Epson Adjustment Program resetter application windows.
  20. Use the printer to print documents as usual.
