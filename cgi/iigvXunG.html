How to Reset an Epson L220 Printer

  1. Download the Epson L220 resetter application.
  2. Extract the downloaded file, which is still in RAR format.
  3. Open the file named Adjprog until the resetter application window appears.
  4. Check the Model section on the left side to see whether your printer model is displayed or if it shows Not Selected.
  5. If the printer model is not selected, click the Select menu to open the Select window and choose the printer model and port.
  6. Select the printer type by clicking the Model Name field.
  7. Select the printer Port that shows your printer series after the port name, or choose Auto Selection to let the application automatically detect the printer port.
  8. Click OK; the Select window will close.
  9. Click the Particular Adjustment Mode menu on the right side; a window containing a list of Adjustment and Maintenance options will appear.
  10. Select Waste Ink Pad Counter under the Maintenance section.
  11. Click OK; the Waste Ink Pad Counter window will open.
  12. Check the Main Pad Counter option.
  13. Click the Check button under Check the counter value.
  14. Wait until the loading process is complete; the previously empty main pad counter value will be displayed.
  15. Check the Main Pad Counter option again.
  16. Click the Initialize button under Initialize the selected counters.
  17. Click OK when the Information dialog appears to confirm the process.
  18. Click OK again if a message appears instructing you to turn off the printer.
  19. Turn off the printer by pressing and holding the power button until the power indicator light turns off.
  20. Click OK if a dialog appears instructing you to turn the printer back on.
  21. Turn the printer back on by pressing the power button once; if the power indicator light does not blink, the reset process has been successfully completed.
