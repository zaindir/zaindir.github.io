How to Reset an Epson L3100 Printer

  1. Download the resetter file for the Epson L3001 printer series.
  2. Temporarily disable your antivirus software.
  3. Extract the downloaded RAR file.
  4. Open the extracted folder.
  5. Open the Adjprog file.
  6. Copy the license from the License ID for registration section.
  7. Go back to the extracted folder and open the Keygen file.
  8. Paste the copied license into License ID, then click New Key.
  9. Copy the code under Activation Key.
  10. Open the Adjprog application, then click Enter Key.
  11. Paste the copied code into Activation Key for removal of restrictions and click OK. The application is now ready to use.
  12. Turn on and connect the printer to the laptop or PC being used.
  13. Open the Adjprog file.
  14. Click the Select menu.
  15. Select L3100 as the model name.
  16. Leave the port on Auto Selection.
  17. Click OK.
  18. Click the Particular Adjustment Mode menu.
  19. Select Waste Ink Pad Counter under the Maintenance section.
  20. Check the Main Pad Counter option.
  21. Click the Check button below; the main pad counter value will be displayed.
  22. Click Initialize; an Information dialog will appear.
  23. Click OK; another Information dialog will instruct you to turn off the printer.
  24. Turn off the printer by pressing and holding the power button until the indicator light turns off.
  25. Turn the printer back on; it should now operate normally.
