How to Reset an Epson 1390 Printer

  1. Connect the printer to be reset to a laptop or PC using a USB cable.
  2. Change the date on your PC or laptop to October 4, 2007.
  3. Extract the downloaded RAR file.
  4. Open the extracted folder.
  5. Right-click the Adjprog file and select Properties, then open the Compatibility tab. Check Run this program in compatibility mode for, select Windows XP (Service Pack 2), then click OK.
  6. Right-click the Adjprog file again and select Run as administrator.
  7. Click Accept.
  8. Click the Select button.
  9. In the Port section, select the USB port connected to the printer, usually indicated by the name Stylus Photo 1390, then click OK.
  10. Click Particular Adjustment Mode.
  11. Click Waste Ink Pad Counter.
  12. Check the 0 column and 80 column options, then click Initialization and click OK.
  13. Turn off the printer when the message Please turn off the printer appears, then click OK.
  14. Turn the Epson Stylus Photo 1390 printer back on; the printer is now ready to be used normally again.
