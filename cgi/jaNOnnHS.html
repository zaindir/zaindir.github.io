How to Reset an Epson L1110 Printer

  1. Download the Epson L1110 resetter application.
  2. Extract the downloaded RAR file, then open the extracted folder.
  3. Open the EPSON L1110 RESETTER folder from the extracted files.
  4. Open the Adjprog application; a warning stating that the Hardware ID has been copied will appear. Close the application.
  5. Open the KEY L1110 folder from the extracted files, then open the EXE folder.
  6. Open the WLGen_L1110 application.
  7. Click License Manager, then click Add License.
  8. Enter your name in both the Customer and Company fields.
  9. Paste the Hardware ID into the Hardware ID field (or press Ctrl + V), then click Save. The license name you created will appear.
  10. Click Create License Key.
  11. Open the KEY L1110 folder again, then open the EXE folder; a new folder will appear with the license name you created.
  12. Open the newly created folder and copy all files inside (usually there are five files).
  13. Paste the copied files into the EPSON L1110 RESETTER folder. The resetter application is now ready to use.
  14. Open the Adjprog application from the extracted files.
  15. If a Security Warning dialog appears, click Run; the EPSON Adjustment Program window will open.
  16. Click the Select menu at the top right; you will be prompted to enter the printer model and port.
  17. Ensure the printer model in the Model Name field is correct. If it is empty or not shown, click the field and select L1110.
  18. Click the Port field to select the port connected to the printer.
  19. Select the port that ends with the printer series name, for example USB001 (L1110 Series).
  20. Click OK to return to the main EPSON Adjustment Program window.
  21. Click the Particular Adjustment Mode menu at the bottom right; a dialog box with adjustment options will appear.
  22. Select Waste Ink Pad Counter under the Maintenance section.
  23. Click OK; the Particular Adjustment Mode dialog will close.
  24. Check the Main Pad Counter option.
  25. Also check the Platen Pad Counter option.
  26. Click the Check button next to Check the counter value; the points and percentages will be displayed, and the checkmarks will be cleared.
  27. Check the Main Pad Counter and Platen Pad Counter options again.
  28. Click the Initialize button next to Initialize the selected counters; an Information dialog box will appear.
  29. Click OK; another dialog will appear instructing you to turn off the printer.
  30. Click OK on the dialog.
  31. Turn off the printer by pressing and holding the power button until the indicator light turns off.
  32. Click OK again if another dialog box appears.
  33. Turn the printer back on by pressing the power button.
  34. Click the Finish button.
  35. Close the EPSON Adjustment Program window; the printer is now ready to be used again.
