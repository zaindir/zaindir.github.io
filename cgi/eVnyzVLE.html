How to Reset an Canon MP258 Printer

  1. Make sure the printer is powered off.
  2. Press and hold the Stop/Reset button on the printer and do not release it.
  3. Press and hold the Power button while still holding the Stop/Reset button.
  4. Press the Stop/Reset button five times. On the fifth press, press and hold the Stop/Reset button again without releasing it, so that both the Power and Stop/Reset buttons are held simultaneously.
  5. Release both the Stop/Reset and Power buttons at the same time.
  6. If the printer still shows an error or the method does not work, turn the printer off again.
  7. Press and hold the Stop/Reset button.
  8. Press and hold the Power button while continuing to hold the Stop/Reset button.
  9. Press the Stop/Reset button two times. On the second press, press and hold the Stop/Reset button without releasing it, so that both buttons remain pressed.
  10. Release both the Stop/Reset and Power buttons simultaneously.
  11. Wait for a few moments. If successful, the printer will enter service (safe) mode, indicated by the printer becoming active and the number 0 appearing on the printer’s display.
  12. Close any notification windows if the computer detects a new printer or device.
  13. Download the Canon Service Tool v3400 resetter application.
  14. Extract the downloaded resetter if it is still in ZIP format.
  15. Open the Service Tool application.
  16. Prepare blank paper and place it into the printer’s paper tray.
  17. Select Main in the Absorber field under Clear Ink Counter.
  18. Click the Set button next to Absorber – Clear Ink Counter.
  19. Wait until the process is complete; the printer will automatically print a document.
  20. Click OK if a dialog box appears with the message “A function was finished”.
  21. Ensure Main is selected in the Absorber field under Ink Absorber Counter.
  22. Click the Set button next to Absorber – Ink Absorber Counter.
  23. Click OK again if the message “A function was finished” appears.
  24. Click the EEPROM button in the Print section; a Paper Source dialog will appear.
  25. Select Rear Tray in the Paper Source column.
  26. Click OK; the printer will automatically print another document.
  27. Close all windows of the Canon Service Tool v3400 application.
  28. Turn off the printer.
  29. Turn the printer back on; the printer should now be free from errors and function normally again.
