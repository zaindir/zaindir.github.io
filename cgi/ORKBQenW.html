How to Reset an Epson L360 Printer

  3. Extract the downloaded RAR file.
  4. Open the application by right-clicking the Adjprog file, then select Run as Administrator. The application window will open.
  5. Click the Select menu at the top right; you will be asked to enter the printer model and port.
  6. Click the Model Name field to display a list of Epson printer models.
  7. Select the printer model to be reset, in this case Epson L360.
  8. Click the Port field to display the available printer ports.
  9. Select the port that ends with your printer series name, for example USB006 (L360 Series).
  10. Click OK to confirm.
  11. Click the Particular Adjustment Mode menu at the bottom right of the window to open the list of available maintenance and adjustment options.
  12. Select Waste Ink Pad Counter under the Maintenance section.
  13. Click OK; the Waste Ink Pad Counter window will open.
  14. Check the box next to Main Pad Counter.
  15. Click the Check button under Check the current counter value; the checkmark will disappear and the counter values (points and percentages) will be displayed.
  16. Check the Main Pad Counter box again.
  17. Click the Initialize button under Initialize the selected counters; an Information dialog box will appear.
  18. Click OK.
  19. Wait until the loading process is complete.
  20. Click OK if an Error dialog box appears.
  21. Click the Initialize button once again under Initialize the selected counters.
  22. Click OK if an Information dialog box appears.
  23. Wait until another Information dialog appears instructing you to turn off the printer.
  24. Click OK.
  25. Turn off the Epson L360 printer by pressing and holding the power button until the indicator light turns off.
  26. Turn the printer back on.
  27. Close the resetter application; the printer is now ready to be used for printing documents again.
