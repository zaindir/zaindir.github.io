How to Reset an Epson L110 Printer

  1. Make sure the printer is turned on and connected to the computer using the printer USB cable.
  2. Extract the downloaded application.
  3. Open the AdjProg file.
  4. Click the Select button on the main page of the application.
  5. Select the printer model you want to reset in the Model Name column; in this case, L110.
  6. Click OK.
  7. Click Particular Adjustment Mode; a list of adjustment and maintenance options will appear.
  8. Select Waste Ink Pad Counter under the Maintenance section.
  9. Click OK; the Waste Ink Pad Counter window will open.
  10. Click the Check button under The current counter value is confirmed; the points and percentage will be displayed.
  11. Check the Main Pad Counter option.
  12. Click the Initialization button under Initialization will clear the selected above counters.
  13. Click OK if a notification message appears.
  14. Wait for the reset process to complete until a message instructs you to turn off the printer.
  15. Turn off the printer by pressing the power button.
  16. Click OK on the message that appears; another message will instruct you to turn the printer back on.
  17. Turn on the printer.
  18. Click OK on the message.
  19. Wait until the printer is fully on and the indicator lights stop blinking; the printer reset is now complete.
  20. Close all application windows.
  21. Use the printer as usual.
