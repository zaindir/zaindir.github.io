How to Reset an Epson L5290 Printer

  1. Turn on the printer and connect it to the computer.
  2. Download the Epson L5290 resetter application from the internet. One of the resetter applications that can be used is the Epson Adjustment Program.
  3. Open the folder containing the downloaded resetter application.
  4. Extract the resetter application files first.
  5. If a password is required, copy the application password by opening the Notepad file that contains the password.
  6. Double-click the installation file or the file named Adjprog.
  7. Wait a moment until the password window appears.
  8. Paste the password into the provided field.
  9. Click OK; the resetter application will open.
  10. Click the Select menu.
  11. Select the printer model to be reset in the Model Name section; in this case, choose L5290.
  12. Click OK.
  13. Click the Particular Adjustment Mode menu; several options will be displayed.
  14. Select Waste Ink Pad Counter under the Maintenance options.
  15. Click OK; the Waste Ink Pad Counter window will open.
  16. Make sure all options are checked, especially Main Pad Counter, to check the printer usage points and percentage.
  17. Click the Check button; the points and percentages for the selected items will be displayed.
  18. Check the options again, except for the middle option or Platen Pad Counter, which can be left unchecked.
  19. Click the Initialize button.
  20. Click OK if an Information dialog box appears.
  21. Wait until a dialog box appears instructing you to turn off the printer.
  22. Turn off the printer.
  23. Click OK in the dialog box.
  24. Click OK again when a dialog box appears indicating that the waste ink pad counter has been successfully initialized.
  25. Click Finish.
