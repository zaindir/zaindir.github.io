How to Reset an Epson L380 Printer

  1. Make sure the printer is turned on and do not forget to connect the printer to the computer using the printer USB cable.
  2. Download the Epson L380 printer resetter application.
  3. Open the folder containing the downloaded application.
  4. Right-click the Run-ME file.
  5. Select Run as administrator.
  6. Click OK if an Unregistered version window appears; after that, the Epson Adjustment Program resetter application window will be displayed.
  7. Click the Select button.
  8. Choose the printer model you want to reset in the Model Name column, namely L380.
  9. Select the printer connection in the Port section, for example USB002 (L380 Series) if the printer is connected via a USB cable.
  10. Click OK.
  11. Click the Particular Adjustment Mode menu; various options will appear.
  12. Select Waste Ink Pad Counter under the Maintenance section.
  13. Click OK, then the Waste Ink Pad Counter window will open.
  14. Check the Main Pad Counter option.
  15. Click the Check button, or you can directly click Initialize to reset the printer.
  16. Click OK if an Information dialog box appears indicating that the reset process is ready to proceed.
  17. Wait a few moments, then click OK again when another Information dialog box appears.
  18. Turn off the printer.
  19. Turn the printer back on; the Epson L380 printer has now been successfully reset.
  20. Close the resetter application window, and the printer is ready to be used again.
