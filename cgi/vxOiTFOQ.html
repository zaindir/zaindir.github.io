How to Reset an Epson L120/L121 Printer

  1. Turn on the Epson L120/L121 printer that you want to reset.
  2. Make sure the printer is connected to the computer or laptop used for resetting.
  3. Extract the downloaded resetter files.
  4. Double-click the extracted exe file.
  5. Click the Select button.
  6. In the Port section, select the USB port being used, for example USB002 (L120/L121 Series).
  7. Click OK.
  8. Click Particular Adjustment Mode; a list of adjustment and maintenance options will appear.
  9. Select Waste Ink Pad Counter under the Maintenance section.
  10. Click OK; the Waste Ink Pad Counter window will be displayed.
  11. Check the Main Pad Counter option.
  12. Click the Check button under Check the current counter value; the points and percentage of the main pad counter will be displayed.
  13. Check the Main Pad Counter option again.
  14. Click the Initialize button under Initialize the selected counters to start resetting the printer.
  15. Click OK if a notification message appears.
  16. Wait a few moments until a dialog box appears with instructions to turn off the printer.
  17. Turn off the printer by pressing the power button.
  18. Click OK on the message that appears.
  19. Click Finish and close all resetter application windows.
  20. Turn the printer back on and try printing a document; if it prints without issues, the printer has been successfully reset.
