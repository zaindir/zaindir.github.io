How to Reset an Epson L5190 Printer

  1. Turn on the Epson L5190 printer and connect it to the laptop or PC that will be used for resetting.
  2. Temporarily disable any antivirus software on your computer.
  3. Extract the downloaded RAR resetter file.
  4. Open the extracted folder; there will be two folders: app and key.
  5. Open the app folder.
  6. Right-click the Adjprog file and select Run as administrator. An error message showing your Hardware ID will appear. Do not close this window yet.
  7. Open the key folder, then open the EXE subfolder.
  8. Right-click WLGEN_Epson L5190 and select Run as administrator. The Key Generator application will open.
  9. Click License Manager, then click Add License.
  10. Enter any name in the Customer field, for example aa. Do the same for the Company field.
  11. Enter the Hardware ID from the error message shown in step 6.
  12. Click Save, then click Create License Key and confirm by clicking OK. A new folder will appear in File Explorer with the customer and company name, for example aa-aa
  13. Open the new folder created by the Key Generator (e.g., aa-aa) and copy all the files inside.
  14. Paste the copied files into the folder where Adjprog is located, which is the app folder.
  15. Right-click Adjprog and select Run as administrator. Click OK if the WinLicense message appears. The resetter application will open.
  16. Click Particular Adjustment Mode.
  17. Select Waste Ink Pad Counter and click OK.
  18. Check both the Main Pad Counter and Platen Pad Counter options, then click Check to view the printer status.
  19. Check the Main Pad Counter and Platen Pad Counter again, then click Initialize and OK. Wait until the process completes.
  20. Turn off the printer when the message Please turn off the printer appears, then click OK.
  21. Turn the printer back on; the Epson L5190 is now ready to use.
