How to Reset an Canon G2000 Printer

  1. Prepare several sheets of plain paper and place them into the printer’s paper tray.
  2. Make sure the printer is powered off.
  3. Press and hold the Stop button on the printer and do not release it.
  4. Press and hold the Power button while still holding the Stop button, so that both buttons are pressed simultaneously.
  5. Release the Stop button only, while continuing to hold the Power button.
  6. Press the Stop button five times. On the fifth press, press and hold the Stop button again without releasing it.
  7. Release both the Stop and Power buttons at the same time. The printer will turn on, the green Power indicator will blink, and the red Stop indicator will light up.
  8. Open the top cover of the printer until the print head is visible to deactivate the red Stop indicator.
  9. Close the top cover again; the red Stop indicator should now turn off.
  10. Wait until the green Power indicator stops blinking and remains steady. The printer has now entered service (safe) mode and is ready to be reset.
  11. Download the Canon G2000 resetter application.
  12. Extract the downloaded file.
  13. Open the extracted folder and run the ST4905 application.
  14. Select Main in the Absorber field under Ink Absorber Counter.
  15. Click the Set button next to the Counter value field.
  16. Click OK if a dialog box appears with the message “A function was finished”.
  17. Click the EEPROM button in the Print section; a Paper Source dialog will appear.
  18. Select Rear Tray in the Paper Source column.
  19. Click OK; the printer will automatically perform a test print.
  20. Select Main in the Absorber field under Clear Ink Counter.
  21. Click the Set button aligned with Absorber – Clear Ink Counter.
  22. Click Close if the EEPROM Information window appears.
  23. Click the EEPROM button again in the Print section.
  24. Select Rear Tray again in the Paper Source column.
  25. Click OK; the printer will automatically print another document.
  26. Close the Canon Service Tool v4905 application.
  27. Turn off the printer, then turn it back on. The printer is now ready to be used normally.
