How to Reset an Epson L310 Printer

  1. Download the Epson L310 printer resetter application.
  2. Extract the downloaded file until the resetter file named Adjprog is available.
  3. Open the Adjprog application.
  4. Click the Select menu if the Model section is not filled or shows Not Selected; the Select dialog box will open.
  5. Select the printer model in the Model Name section according to the printer being used, namely Epson L310.
  6. Select the printer Port that shows your printer model or series after the port name.
  7. Click OK; the dialog box will close and the printer model will appear in the Model section.
  8. Click the Particular Adjustment Mode menu; a window containing various Adjustment and Maintenance options will appear.
  9. Select Waste Ink Pad Counter.
  10. Click OK; the Waste Ink Pad Counter dialog box will open.
  11. Check the box next to Main Pad Counter.
  12. Click the Check button next to Check the current counter value; the main pad counter points and percentage will be displayed.
  13. Check the Main Pad Counter option again.
  14. Click the Initialize button next to Initialize the selected counters; an Information dialog box will appear.
  15. Click OK.
  16. Wait until the loading process is complete.
  17. Click OK again if another Information dialog appears instructing you to turn off the printer.
  18. Turn off the printer by pressing and holding the power button for a few seconds until the indicator light turns off.
  19. Close all resetter application windows.
  20. Turn the printer back on by pressing the power button once; the printer is now back to normal and ready to print documents.
